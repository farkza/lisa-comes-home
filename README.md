# 🍑 Lisa Comes Home — GR20 Countdown

Une petite application web pour compter les jours jusqu'au retour de Lisa de son GR20, avec un **journal de bord interactif** pour suivre chaque étape de sa randonnée.

## ✨ Fonctionnalités

- **Compteur de jours** — combien de jours avant que Lisa rentre 🏠
- **Jauge de progression** — visualisation du chemin parcouru sur les 16 étapes
- **Journal de bord** — Lisa peut inscrire chaque jour :
  - Kilomètres parcourus
  - Dénivelé réalisé
  - Notes / humeur du jour
- **Persistance locale** — tout est sauvegardé dans le `localStorage`

## 📅 Dates du GR20

- **Départ** : 4 juin 2026
- **Retour** : 19 juin 2026
- **Durée** : 16 étapes officielles (Nord → Sud)

- [Tailwind CSS](https://tailwindcss.com) — Styling
- [shadcn/ui](https://ui.shadcn.com) — Composants UI
